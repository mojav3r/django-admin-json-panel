## Description
Custom Django Admin Panel for Research Institute

## Installation

1. Install dependencies:
```
pip install -r requirements.txt
```
3. Make migrations:
```
python manage.py makemigrations
```
3. Run migrations:
```
python manage.py migrate
```
4. Run the project:
```
python manage.py runserver 8000
```
